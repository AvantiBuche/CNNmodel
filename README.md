## Dataset
This dataset has 15 labels with 3991 colored images.

The labels are: Bag, Blazer, Cap, Dress, Jwellery, Outwear, Pants, Scarf, Shoes, Shorts, Skirt, Sunglasses, T-Shirt, Top, Vallet.
